# intall procése:

## **installer pyenv avant de commencer ce tutorial d'installation du projet :**
voici un lien d'une documentation pour intaller pyenv sur votre machine
[clicker sur moi](https://ggkbase-help.berkeley.edu/how-to/install-pyenv/)

#### **colonner le projet en local sur votre machine :**

```
    git clone https://github.com/CF-206/Projet_transversal_TPRE241.git
    cd Projet_transversal_TPRE241
```

#### **installer et activer l'environemnt isoler :**

```
    python -m venv venv
    .venv\Scripts\activate
```

#### **intaller les lib et dépendances pour ce projet :**

```
    pip install -r requirements.txt
```
